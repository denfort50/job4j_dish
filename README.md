# job4j_dish

Данный репозиторий наряду с job4j_admin является частью проекта по управлению списком блюд.  
Этот микросервис предоставляет REST API методы для управления блюдами.

## Стек технологий
* Java 17
* PostgreSQL 14
* Apache Maven 3.8.5
* Spring Boot 2.7.10
* Liquibase 4.9.1
* Lombok 1.18.26
* JUnit 5.8.2
* Mockito 4.5.1
* Checkstyle 10.7.0
* Thymeleaf 3.0.15
